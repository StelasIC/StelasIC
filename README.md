## 📌 Sobre o Projeto

Bem-vindas ao projeto **STELAS**! 

O **Projeto STELAS** é uma iniciativa de alcance nacional, focada em utilizar a tecnologia e a telessaúde como instrumentos de inclusão. Nosso objetivo central é estimular o interesse e fortalecer a presença feminina em áreas tecnológicas e científicas, criando um ambiente de aprendizado, desenvolvimento e pertencimento.

## 💻 O Curso CITE

Como uma das principais ações do projeto, oferecemos o **CITE (Curso Introdutório de Tecnologias Emergentes)**. 

O curso apresenta conceitos fundamentais sobre as tecnologias que estão moldando o futuro, estimulando a participação feminina na ciência e na tecnologia de forma prática e acessível. Os módulos cobrem desde o pensamento computacional até o desenvolvimento de projetos finais aplicados à realidade das alunas.

## ⚙️ Arquitetura e Dados (Backstage do Projeto)

Para sustentar as análises, métricas de impacto e o mapeamento de dados do projeto, nossa infraestrutura de pesquisa e engenharia de dados utiliza:

*   **Linguagens:** Python e R.
*   **Visualização e Dashboards:** Desenvolvimento de painéis interativos utilizando Shiny para monitoramento dos indicadores de inclusão.
*   **Foco de Pesquisa:** Engenharia de Dados, Ciência de Dados e aplicações voltadas à cibersegurança e proteção de dados no contexto de tecnologias sociais.

## 🤝 Como Contribuir

Se você faz parte da equipe de desenvolvimento ou tutoria e deseja contribuir com os códigos, materiais do curso ou dashboards do projeto:

1. Faça um *Fork* deste repositório.
2. Crie uma *Branch* para sua modificação (`git checkout -b feature/MinhaContribuicao`).
3. Faça o *Commit* das suas alterações (`git commit -m 'Adicionando novo material ao CITE'`).
4. Faça o *Push* para a Branch (`git push origin feature/MinhaContribuicao`).
5. Abra um *Pull Request*.
